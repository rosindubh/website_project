To launch click index.html

The  <A Little Bit About Me>  link on the homepage will open the 
aboutMe.html file in a new tab.

The <Github Setup> link at the top of the homepage will open a new page
in a new tab giving instructions how to install Git on a Linux system.

In section 1.3 of the Homepage there is a link to Team4s trello board.

In section 1.6 of the Homepage you will find a link to Github 
which will open in a seperate tab and display the githubSetup.html file.
There is a link to Github on this page too.
